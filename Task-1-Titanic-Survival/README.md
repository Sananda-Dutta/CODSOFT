# Titanic Survival Prediction Project

## What Does This Do?
- Predicts if a passenger survived the Titanic using their:
  - Age
  - Gender (Male/Female)
  - Ticket class (1st/2nd/3rd)
- Accuracy: 83% (8 out of 10 predictions correct)
